# Random unit Defense

## 게임 컨셉

## High Concept
랜덤으로 유닛을 소환하고, 스킬을 활용하여 적의 웨이브를 막는 타워 디펜스 게임

## 핵심 메카닉
적이 정해진 경로에 따라 움직이고, 포탑이 범위 안에 오면 자동 공격한다.

---

## 예상 게임 흐름

1. 플레이어가 코스트를 사용하여 유닛을 랜덤하게 뽑는다.
2. 적이 일정 시간마다 정해진 경로에 따라 움직인다.
3. 범위안에 들어오면 자동공격하는 유닛으로 적을 막는다.
4. 웨이브가 끝난 후 일정 코스트를 받는다.
5. 코스트로 유닛을 강화하거나 유닛을 뽑는다.
6. 일정 웨이브 후 보스가 나와 보스를 잡으면 승리한다.

## 개발 범위

맵: 1개
적 종류: 4종
  일반 적
  빠른 적
  체력이 높은 적
  보스

유닛 종류: 4종
  일반 유닛
  희귀 유닛
  유니크 유닛
  전설 유닛

스킬: 2종
  전체 공격 스킬
  일정 시간 공격력 증가 스킬

웨이브: 5단계

## 개발 일정 (8주)

- 1주차: 프로젝트 설정 및 기본 구조 설계
- 2주차: 맵 및 자원, 유닛 소환 구현
- 3주차: 유닛 공격 및 랜덤 시스템 구현
- 4주차: UI 구현
- 5주차: 적 이동 시스템 및 체력 구현
- 6주차: 충돌 처리 구현
- 7주차: UI 및 화면 구성
- 8주차: 테스트 및 최종 수정
